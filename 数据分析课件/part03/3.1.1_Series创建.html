3.1.1 Series创建

Series对象本质上是一个NumPy的数组，因此NumPy的数组处理函数可以直接对Series进行处理。每个Series对象实际上都由两个数组组成，他们内部的结构很简单，由两个相互关联的数组组成，其中主数组用来存放数据。主数组的每个元素都有一个与之县关联的标签，这些标签存储在另外一个叫做Index的数组中.

注意三点：

  1. Series是一种类似于一维数组(ndarray)的对象.
  2. 数组中可存储多种数据类型.
  3. 数组中存在索引.

1. Series创建
from pandas import Series
import pandas as pd
import numpy as np

# 没有指定索引，默认1~N作为索引
series1 = Series([10, 20, 30, 40])
series1
#    0    10
#    1    20
#    2    30
#    3    40
#    dtype: int64

# 字典的key作为索引，value作为元素
series2 = Series({'aaa': 10, 'bbb': 20, 'ccc': 30, 'ddd': 40})
series2
#    aaa  
#    bbb  
#    ccc  
#    ddd  
#    dtype: int64

# Numpy数组也可以创建Series, 这里需要额外注意,
# 新的Series中的元素不是原数组的或Series对象元素的副本，而是对他们的引用
ndarray = np.arange(10, 15)
series3 = Series(ndarray)
series3
#    0    10
#    1    11
#    2    12
#    3    13
#    4    14
#    dtype: int64

ndarray[0] = 100
series3
#    0    100
#    1     11
#    2     12
#    3     13
#    4     14
#    dtype: int64

# 标量创建Series
series4 = Series(10)
series4
#    0    10
#    dtype: int64

# 标量创建Series
series5 = Series(10, index=list('abcdef'))
series5
#    a    10
#    b    10
#    c    10
#    d    10
#    e    10
#    f    10
#    dtype: int64

2. Series索引
# 我们可以通过index来访问Series的索引， values来访问Series的值
series4 = Series(np.arange(1, 6))
series4
#    0    1
#    1    2
#    2    3
#    3    4
#    4    5
#    dtype: int64

# 输出Series的索引列表
series4.index
#    RangeIndex(start=0, stop=5, step=1)

# 输出Series的值列表
series4.values
#    array([1, 2, 3, 4, 5])

# Series的索引可以手动指定
series4.index = list('abcde')
series4
#    a    1
#    b    2
#    c    3
#    d    4
#    e    5
#    dtype: int64

# 也可以在创建的时候指定索引
series5 = Series(np.arange(10, 60, 10), index=list('abcde'))
series5
#    a    10
#    b    20
#    c    30
#    d    40
#    e    50
#    dtype: int64
