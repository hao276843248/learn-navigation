3.6.2 读写HTML文本文件数据

Pandas提供了一下I/O API函数用于读写HTML格式的文件:

read_html to_html

这两个函数非常有用. 把DataFrame等复杂的数据结构转换为HTML表格很简单,无需编写一长串的HTML代码就能实现.

如今主要的数据源为互联网.网上的数据并不是存储在TXT或者CSV文件中，而是网页文本的一部分，因此pandas也可以读取网页数据. 读取网页数据这种操作我们叫做网页抓取，应用极广. 它逐渐演变为数据分析过程中的一项基础操作, 被整合到了数据分析的第一步-数据准备.

to_html函数会将pandas对象转变成html代码, 我们需要将html代码写入html文件.

read_html函数会解析html页面, 寻找HTML表格. 如果找到HTML表格就将其转换为可以直接用于数据分析的DataFrame对象.

注意:

  1. 如果报错:ImportError: lxml not found, please install it 执行命令: pip install lxml
  2. 如果报错:ImportError: BeautifulSoup4 (bs4) not found, please install it 执行命令: pip install beautifulsoup4
