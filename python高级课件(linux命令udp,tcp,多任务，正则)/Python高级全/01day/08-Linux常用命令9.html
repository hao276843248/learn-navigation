Linux 终端命令(gzip)

1.文件压缩解压：gzip

tar与gzip命令结合使用实现文件打包、压缩。 tar只负责打包文件，但不压缩，用gzip压缩tar打包后的文件，其扩展名一般用xxxx.tar.gz。

gzip使用格式如下：

gzip  [选项]  被压缩(解压)文件

常用选项：

选项 含义
-d 解压

1.1.压缩

# 切换到itheima目录下
parallels@parallels-vm:~/Desktop$ cd itheima/
# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/itheima$ ls
1  1.txt  2.txt  hm.tar
# 对hm.tar包文件进行压缩
parallels@parallels-vm:~/Desktop/itheima$ gzip hm.tar 
# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/itheima$ ls
1  1.txt  2.txt  hm.tar.gz

1.2.解压

# 切换到itheima目录下
parallels@parallels-vm:~/Desktop$ cd itheima/
# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/itheima$ ls
1  1.txt  2.txt  hm.tar.gz
# 对hm.tar.gz压缩包进行解压
parallels@parallels-vm:~/Desktop/itheima$ gzip -d hm.tar.gz 
# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/itheima$ ls
1  1.txt  2.txt  hm.tar

tar这个命令并没有压缩的功能，它只是一个打包的命令，但是在tar命令中增加一个选项(-z)可以调用gzip实现了一个压缩的功能，实行一个先打包后压缩的过程。

压缩用法：tar -cvzf 压缩包包名 文件1 文件2 ...

1.3.打包并压缩

# 切换到itheima目录下
parallels@parallels-vm:~/Desktop$ cd itheima/
# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/itheima$ ls
1  1.txt  2.txt
# 把1.txt文件和2.txt完成打包并压缩操作
parallels@parallels-vm:~/Desktop/itheima$ tar -zcvf hm.tar.gz 1.txt 2.txt 
1.txt
2.txt
# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/itheima$ ls
1  1.txt  2.txt  hm.tar.gz

1.4.解压并解包

# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/itheima$ ls
1  1.txt  2.txt  hm.tar.gz
# 对hm.tar.gz压缩文件进行解压并解包到桌面目录下的hm文件夹中
parallels@parallels-vm:~/Desktop/itheima$ tar -zxvf hm.tar.gz -C ~/Desktop/hm/
1.txt
2.txt
# 切换到Desktop目录下hm文件夹
parallels@parallels-vm:~/Desktop/itheima$ cd ~/Desktop/hm/
# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/hm$ ls
1.txt  2.txt
