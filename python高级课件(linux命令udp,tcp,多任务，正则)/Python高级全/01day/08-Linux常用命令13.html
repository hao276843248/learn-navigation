Linux 终端命令(sudo -s,passwd,exit,who,reboot,shutdown)

1. 切换到管理员账号：sudo -s

Ubuntu系统终端下切换到root的简单命令:

# whoami命令显示登录名
parallels@parallels-vm:~$ whoami
# 普通用户
parallels
# 切换到超级用户（也就是root用户）
parallels@parallels-vm:~$ sudo -s
# # whoami命令显示登录名
root@parallels-vm:~# whoami
root
# 退出则返回上一个登陆账号
root@parallels-vm:~# exit
exit
# whoami命令显示登录名
parallels@parallels-vm:~$ whoami
parallels
注意：
parallels@parallels-vm:~$ 中的$ 标识着是一个普通用户
root@parallels-vm:~#中的#表示着是一个超级用户

2.设置用户密码：passwd

在Unix或者Linux中，超级用户可以使用passwd命令为普通用户设置或修改用户密码。普通用户也可以直接使用该命令来修改自己的密码。

parallels@parallels-vm:~$ passwd parallels
更改 parallels 的密码。
（当前）UNIX 密码： 
输入新的 UNIX 密码： 
重新输入新的 UNIX 密码： 
passwd：已成功更新密码

3.退出登录账户： exit

如果是图形界面，退出当前终端；

如果是使用ssh远程登录，退出登陆账户；

如果是切换后的登陆用户，退出则返回上一个登陆账号；

4.查看登录用户：who

who命令用于查看当前所有登录系统的用户信息。

# who命令用于查看当前所有登录系统的用户信息

parallels@parallels-vm:~$ who

parallels tty7 2018-03-03 23:00 (:0)

# 只显示用户的登录账号和登录用户的数量

parallels@parallels-vm:~$ who -q

parallels

用户数=1

# 显示上一次重新启动的时间和日期

parallels@parallels-vm:~$ who -b

     系统引导 2018-03-03 23:00
注意：
parallels tty7 2018-03-03 23:00 (:0)
  • 第一列显示用户名称
  • 第二列显示用户连接方式。tty意味着用户直接连接到电脑上，而pts意味着远程登录。
  • 第三、四列分别显示日期和时间(该用户登录时间)
  • 第五列显示用户登录IP地址

5.关机重启：reboot、shutdown

命令 含义
reboot 重新启动操作系统
shutdown –r now 关机后立即重新启动操作系统
shutdown -h now 立刻关机，其中now相当于时间为0的状态
shutdown -h 20:25 系统在今天的20:25 会关机
shutdown -h +10 系统再过十分钟后自动关机
注意：
reboot就像按了机箱上的重启键一样暴力
shutdown命令可以安全地关闭或重启Linux系统
