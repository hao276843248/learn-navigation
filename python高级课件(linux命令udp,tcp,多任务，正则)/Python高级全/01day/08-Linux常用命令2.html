Linux常用命令(mkdir,tree,touch,rm)

1. 创建目录：mkdir

2.以树状形式显示目录结构：tree

通过mkdir命令可以创建一个新的目录。参数-p可递归创建目录。

需要注意的是新建目录的名称不能与当前目录中已有的目录或文件同名，并且目录创建者必须对当前目录具有写权限。

# 显示当前路径
parallels@parallels-vm:~/Desktop$ pwd
/home/parallels/Desktop
# 在当前目录下创建itheima文件夹
parallels@parallels-vm:~/Desktop$ mkdir itheima
# 切换到itheima目录下
parallels@parallels-vm:~/Desktop$ cd itheima/
# 查看该目录下文件信息
parallels@parallels-vm:~/Desktop/itheima$ ls
# 在当前目录下递归创建文件夹报错
parallels@parallels-vm:~/Desktop/itheima$ mkdir a/b/c/d
mkdir: 无法创建目录"a/b/c/d": 没有那个文件或目录
# 在当前目录下递归创建文件夹报错解决方案加 -p
parallels@parallels-vm:~/Desktop/itheima$ mkdir a/b/c/d -p
# 以目录树的方式显示
parallels@parallels-vm:~/Desktop/itheima$ tree
.
└── a
    └── b
        └── c
            └── d

4 directories, 0 files
# 查看当前目录
parallels@parallels-vm:~/Desktop/itheima$ ls
a

3.创建文件: touch

用户可以通过touch来创建一个空的文件，demo如下：

touch hello.txt

说明：

  • 则会在当前路径下创建名字为hello.txt的空文件
  • Linux系统中没有严格的后缀（格式），所以创建文件时可以命名为任意的文件名

4.删除文件：rm

可通过rm删除文件或目录。使用rm命令要小心，因为文件删除后不能恢复。为了防止文件误删，可以在rm后使用-i参数以逐个确认要删除的文件。

常用参数及含义如下表所示：

参数 含义
-i 以进行交互式方式执行
-f 强制删除，忽略不存在的文件，无需提示
-r 递归地删除目录下的内容，删除文件夹时必须加此参数

4.1.删除文件

# 查看当前目录
parallels@parallels-vm:~/Desktop/itheima$ ls
a
# 创建一个文件
parallels@parallels-vm:~/Desktop/itheima$ touch hm.txt
# 查看当前目录
parallels@parallels-vm:~/Desktop/itheima$ ls
a  hm.txt
# 删除文件 会询问是否真的要删除？
parallels@parallels-vm:~/Desktop/itheima$ rm hm.txt -i
# y代表删除 no代表不删除 
rm：是否删除普通空文件 'hm.txt'？ y
# 查看当前目录
parallels@parallels-vm:~/Desktop/itheima$ ls
a
# 创建一个文件
parallels@parallels-vm:~/Desktop/itheima$ touch hm.txt
# 查看当前目录
parallels@parallels-vm:~/Desktop/itheima$ ls
a  hm.txt
# 删除文件不询问
parallels@parallels-vm:~/Desktop/itheima$ rm hm.txt
parallels@parallels-vm:~/Desktop/itheima$ ls
a

4.2.删除目录(文件夹)

# 查看当前目录
parallels@parallels-vm:~/Desktop/itheima$ ls
a
# 创建一个文件夹
parallels@parallels-vm:~/Desktop/itheima$ mkdir hm
# 查看当前目录
parallels@parallels-vm:~/Desktop/itheima$ ls
a  hm
# 使用rm删除一个文件夹报错
parallels@parallels-vm:~/Desktop/itheima$ rm a
rm: 无法删除'a': 是一个目录
# 使用rm删除一个文件夹需要加上-r
parallels@parallels-vm:~/Desktop/itheima$ rm a -r
# 查看当前目录
parallels@parallels-vm:~/Desktop/itheima$ ls
hm
